Serviços - Exercício 12: Criação do Website

 Nesta página, detalhamos os "serviços" que realizamos para completar o Exercício 12 do Projeto Integrador. Embora não sejamos uma empresa de serviços web tradicional, apresentamos aqui as etapas e processos que seguimos na criação deste website, como se fossem os serviços que oferecemos.

1. Planejamento e Compreensão do Desafio

 Nosso primeiro "serviço" foi o planejamento e aprofundamento no desafio proposto. Entendemos que o Exercício 12 requeria a criação de um website com no mínimo quatro páginas interligadas, demonstrando os conhecimentos de desenvolvimento web aprendidos em aula. Este planejamento inicial foi crucial para definir o escopo do projeto e as etapas seguintes.

2. Estrutura e Design da Navegação do Site

 Em seguida, dedicamos tempo à definição da estrutura do site e ao design da navegação. Planejamos as páginas essenciais (Início, Sobre, Serviços, Contato) e como elas se interligariam através do menu de navegação. Este passo foi fundamental para garantir que o site fosse fácil de usar e que a interligação entre as páginas fosse clara e funcional.

3. Criação do Conteúdo das Páginas

 O próximo "serviço" foi a criação do conteúdo textual para cada página do site. Desenvolvemos os textos para as páginas "Início", "Sobre" e "Serviços" (esta página!), buscando apresentar informações relevantes sobre o Exercício 12 e o Projeto Integrador de forma clara e concisa. O conteúdo foi pensado para explicar o propósito do site e guiar o visitante pelas diferentes seções.

4. Desenvolvimento em HTML: Estrutura e Interligação

 Partimos então para o desenvolvimento em HTML, a base da estrutura do website. Codificamos a estrutura de cada página, utilizando as tags HTML aprendidas em aula para organizar o conteúdo, criar títulos, parágrafos, listas e o menu de navegação. A etapa crucial aqui foi garantir a interligação entre as páginas através dos links Início, Sobre, Serviçoes, Contato, permitindo a navegação fluida entre as seções do site.

5. Estilização com CSS: Design Visual e Apresentação

 Para dar vida ao website, aplicamos o "serviço" de estilização com CSS. Criamos um arquivo CSS (`style.css`) para definir o design visual do site, incluindo fontes, cores, espaçamento, layout e outros aspectos estéticos. O CSS foi essencial para tornar o site mais atraente e profissional, além de garantir uma apresentação visual consistente em todas as páginas.

6. Testes e Verificação da Interligação e Funcionalidade

 Por fim, realizei testes e verificações para garantir que o website funcionasse corretamente. Navegamos por todas as páginas, clicamos nos links do menu para confirmar a interligação e verifiquei se o conteúdo era exibido adequadamente em diferentes navegadores (opcionalmente). Este "serviço" de testes foi importante para assegurar a qualidade e a funcionalidade do site como um todo.
